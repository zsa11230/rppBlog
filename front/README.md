# vue-element-cli

## 上手开发

> 推荐使用 `git bash`

- 克隆项目代码
- 预装好 `node.js v10.16.3+` 安装包
- 如果使用 `vscode` 开发,可使用我们推荐的 `vscode` 配置 你也可以选择自定义配置(命令: `cp .vscode.copy/ .vscode/ -rf`)
- 安装项目的依赖 `npm i` 或使用 `yarn` (需要预装，命令：`npm i -g yarn`)
- 拷贝 `cp .env .env.local`
- 修改开发环境代理 ip 配置 文件在 `.env.local`
- 确认安装依赖无错，即可运行 `yarn serve` 或 `npm run serve`
- 提交使用 `yarn cz` 尽量使用命令提交, 冲突等等可不用
